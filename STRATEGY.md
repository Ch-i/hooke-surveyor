# Hooke Park — Silvicultural Strategy & Integration Plan

## Mission
High-quality timber production + maximum biodiversity across all scales, using the actual terrain (contours, drainage, solar, climate projections) and grounded in 30 peer-reviewed techniques (18 DOIs).

## Core Principles (from research)

1. **NEVER clearfell** — Selective thinning preserves soil C; maintain continuous canopy for microclimate buffering (DOI: 10.1016/j.foreco.2020.118127, 10.1111/geb.12991)
2. **Retain all brash/deadwood** during thinning — whole-tree removal loses 10-20% SOC (DOI: 10.1016/j.foreco.2020.118127)
3. **N-fixers lead** — Alder/birch nurse rings around oak; ECM networks facilitate succession (DOI: 10.1111/nph.16569, 10.1007/s10457-018-0261-3)
4. **Site-specific species** — Match species to slope, TWI, solar, aspect (DOI: 10.1007/s11056-015-9506-4)
5. **Climate-proof** — Sessile oak over pedunculate on dry sites; beech ONLY in sheltered moist positions; cork oak, sweet chestnut viable for 2050+ (DOI: 10.1016/j.baae.2020.04.003)
6. **Coppice funds transition** — Hazel/sweet chestnut 7-25yr rotation provides revenue (DOI: 10.1016/j.landusepol.2019.02.025)
7. **Hedgerows are the highest-evidence intervention** for this bioregion (match: 0.90, DOI: 10.1016/j.landusepol.2019.02.025)

## 7 Management Classes (from CHM + forensics)

### A. Monoculture Conifer Blocks (scots pine, douglas fir, h > 15m)
**Strategy:** Gradual transition via CCF. Thin 30% over 5 years, underplant with broadleaf.
- Yr 1-3: Selective thinning (remove suppressed, poorly formed stems) — retain brash
- Yr 3-7: Underplant with birch + common alder (ECM bridge + N-fixation)
- Yr 7-15: Secondary wave — oak, sweet chestnut, wild cherry in canopy gaps
- Keep 20% conifers as "mother trees" (mature ECM networks, shelter, structural timber)
- **Species palette:** birch, common alder, pedunculate oak, sweet chestnut, holly (understory)
- **Timber yield:** Douglas fir thinnings (construction), transition to oak (80yr)
- **Science:** Techniques #4, #6, #16, #24, #25

### B. Semi-Natural Broadleaf (oak, beech, birch, mixed ages)
**Strategy:** Conserve and enrich. Protect existing trees, add understory diversity.
- Minimal intervention — these are the ecological core
- Enrich understory: hazel coppice, holly, field maple, wild service
- Add ground flora: bluebell, wood anemone, wild garlic (ancient woodland indicators)
- Deer management to allow natural regeneration
- Monitor via Sentinel-2 + LiDAR change detection
- **Species palette:** hazel, holly, field maple, bluebell, wood anemone
- **Timber yield:** Selective oak thinning for quality (straight stems, no knots)
- **Science:** Techniques #7, #17, #20, #21, #29

### C. Open Ground / Cleared Areas (h < 2m, gaps)
**Strategy:** Syntropic establishment. Legume soil prep → pioneer nurse → secondary canopy.
- Yr 0: Legume green manure (Lotus corniculatus, trefoil) — 1yr soil prep
- Yr 1: Miyawaki-density pioneer planting (birch, alder, willow) + shrub layer (hawthorn, blackthorn)
- Yr 3: Secondary species into nurse canopy (oak, cherry, hazel)
- Yr 7+: Climax species where nurse canopy provides shelter
- **Species palette:** Lotus corniculatus (soil prep), birch, common alder, hawthorn, then oak, cherry
- **Timber yield:** Fast pioneer biomass (mulch, firewood), then oak standards
- **Carbon:** 3.1-4.5 tCO2/ha/yr in first 20 years (DOI: 10.1073/pnas.1710465114)
- **Science:** Techniques #1, #15, #22, #30

### D. Transition Zones (conifer being underplanted)
**Strategy:** Continuous Cover Forestry. Maintain canopy while shifting species.
- Thin conifers gradually (not >30% in any 5yr period)
- Underplant N-fixers first (alder), then timber broadleaf (oak)
- Alley-cropping pattern: retain conifer rows, plant broadleaf between
- Retain brash as mulch layer — drives soil C redistribution from floor to mineral
- **Species palette:** common alder, birch (pioneer), then oak, sweet chestnut
- **Timber yield:** Conifer thinnings now, broadleaf standards in 50yr
- **Science:** Techniques #3, #6, #16, #18, #19

### E. Riparian / Wet Corridors (TWI > 5, streams, hollows)
**Strategy:** N-fixer riparian buffers. Bank stability + water quality + habitat.
- Common alder dominant (Frankia N-fixation, wet-tolerant, bank-binding roots)
- Goat willow, white willow on saturated sites
- Swamp cypress for experimental deep-wet positions
- Understorey: meadowsweet, comfrey, angelica (nectar + bank stabilization)
- **Species palette:** common alder, goat willow, white willow, swamp cypress
- **Timber yield:** Alder (coppice for charcoal/biochar), willow (biomass)
- **Science:** Techniques #19, #28 (riparian buffers highest evidence in UK)

### F. Ridgeline / Exposed (slope > 25°, high elevation, SW exposure)
**Strategy:** Shelter + drought resilience. Wind-hardy species, deep roots.
- Scots pine retained (most drought-tolerant: 0.7)
- Sessile oak (more drought-tolerant than pedunculate: 0.5 vs 0.4)
- Hawthorn/blackthorn shelter belts on windward edges
- Cork oak experimental plantings (drought 0.9, climate 2050)
- **Species palette:** scots pine (retain), sessile oak, sweet chestnut, hawthorn, cork oak
- **Timber yield:** Scots pine (construction), sessile oak (premium)
- **Science:** Techniques #27, #29

### G. Estate Boundaries / Path Edges
**Strategy:** Dense hedgerows (highest bioregion match: 0.90).
- 3-row mixed hedgerow: hawthorn + blackthorn + field maple
- Internal: hazel, dog rose, spindle, guelder rose
- Standard trees at 8m spacing: oak, wild cherry, field maple
- Provides: pollinator corridor, bat flyways, bird nesting, wind shelter
- **Species palette:** hawthorn, blackthorn, hazel, field maple, dog rose, wild cherry
- **Timber yield:** Hedgerow lays (traditional craft), hazel poles
- **Science:** Techniques #12, #28

## Implementation Phases

### Phase 1: Assessment + Soil Prep (Year 0-1)
- [ ] Run forensics on every hex: classify into A-G management classes
- [ ] Map monoculture blocks from CHM (where height variance < 2m in 50m radius)
- [ ] Identify physarum corridors for priority hedgerow/riparian routes
- [ ] Begin legume soil prep in Class C open ground
- [ ] Initiate deer management plan
- [ ] Baseline soil carbon sampling (10 points per management class)

### Phase 2: Pioneer + N-Fixer Establishment (Year 1-3)
- [ ] Selective thinning of Class A conifer blocks (30% removal, retain brash)
- [ ] Plant N-fixer nurse rings in Class A gaps (alder, birch)
- [ ] Miyawaki pioneer planting in Class C cleared areas
- [ ] Establish riparian buffers along all streams (Class E)
- [ ] Plant hedgerows along estate boundaries (Class G)
- [ ] Revenue: conifer thinnings (construction timber), pioneer biomass (firewood)

### Phase 3: Secondary + Timber Species (Year 3-7)
- [ ] Underplant oak, sweet chestnut, wild cherry in Class A/D canopy gaps
- [ ] Introduce hazel coppice in Class B understory
- [ ] Plant sessile oak on Class F ridgelines
- [ ] Establish ground flora under established canopy (bluebell, garlic, anemone)
- [ ] Continue selective conifer thinning (Class A/D)
- [ ] Revenue: first hazel coppice harvest (yr 7)

### Phase 4: Climax + Diversification (Year 7-15)
- [ ] Beech in sheltered moist Class B sites ONLY (drought vulnerability)
- [ ] Holly, yew understory in mature Class B
- [ ] Wild service, small-leaved lime on best sites
- [ ] Cork oak experimental plots on Class F dry sites (climate 2050)
- [ ] Revenue: sweet chestnut coppice (25yr rotation starting)

### Phase 5: Natural Succession (Year 15-50)
- [ ] Reduce interventions — let ECM networks drive species establishment
- [ ] Monitor via LiDAR + satellite (change detection every 2 years)
- [ ] Selective quality thinning of oak (promote straight stems, reduce knots)
- [ ] Revenue: first oak quality thinnings (yr 30+)

### Phase 6: Mature Resilient Forest (Year 50-100)
- [ ] Mixed-age, multi-species, multi-strata woodland
- [ ] Self-sustaining ECM networks connecting all species
- [ ] Revenue: premium hardwood timber (oak, sweet chestnut, wild cherry)
- [ ] Carbon: 100-200 tC/ha accumulated
- [ ] Biodiversity: NVC W8/W10/W16 communities re-established
